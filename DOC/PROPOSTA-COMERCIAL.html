SICAR API - Proposta Comercial

Descrição

Sistema de automação para download de dados do SICAR (Cadastro Ambiental Rural) com interface web e API REST.

Stack: Python + FastAPI + React + PostgreSQL

Principais funcionalidades:

  • Download automático por estado (27 estados brasileiros)
  • Download individual por número CAR (funcionalidade exclusiva desenvolvida)
  • Resolução automática de CAPTCHA
  • Agendador configurável (diário/semanal/intervalo)
  • Dashboard web com estatísticas
  • API REST com 22 endpoints documentados
